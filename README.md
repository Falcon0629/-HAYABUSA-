# 隼カフェ 〜HAYABUSA〜

架空のカフェを想定し、初来店のユーザーが「どんなお店か」「自分に合いそうか」を一目で判断できるように設計したサイト。

## Live（動くサイト）
- (https://warm-shortbread-2d2356.netlify.app/)

## 目的
- 初見でも迷わず、来店イメージが自然に湧く体験をつくる
- 雰囲気・魅力・行動（来店/確認）の流れを、短時間で理解できる構成にする

## 実装ハイライト
- ファーストビューで「雰囲気 → 提供価値 → 行動」の順に情報を配置し、視線の流れを整理
- 余白・タイポグラフィ・写真の見せ方を調整し、空気感が直感的に伝わるUIに設計
- レスポンシブ対応で、スマホでも読みやすく崩れないレイアウトに調整

## 使用技術
- Figma / HTML / CSS / JavaScript

## AIの使い方（透明性）
- （必要に応じて）文章の推敲や構成整理の補助にAIを使用
- 設計意図の決定・UI設計・実装・調整は自分で実施
